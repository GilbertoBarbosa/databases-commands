## Apache hive data warehouse software
https://hive.apache.org/

Create database
```
> create database <name>;
```

Create table
```
> create table <tableName (field1 type1, field2 type2, ...)>;
```

Show databases
```
> show databases;
```

Show tables
```
> show tables;
```

Use database
```
> use <databaseName>;
```

Describe table
```
> describe <tableName>;
```

Load Data
```
> LOAD DATA LOCAL INPATH 'file' OVERWRITE INTO TABLE <table>;
```
